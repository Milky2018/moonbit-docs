
## Suggestion

```{literalinclude} /sources/error_codes/E4000_fixed/top.mbt
:language: moonbit
```

Or remove the duplicate type parameter if you meant to use the same type:

```{literalinclude} /sources/error_codes/E4000_fixed/top_1.mbt
:language: moonbit
```
